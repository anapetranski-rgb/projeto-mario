🍄 Encanadores Mario & Luigi - Landing Page
Landing Page temática desenvolvida como parte dos desafios do DevClub, focada no universo dos encanadores mais famosos do Reino dos Cogumelos! O projeto consiste em uma página única (One Page) moderna, totalmente responsiva e interativa.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica da página.

CSS3: Estilização personalizada, uso de Flexbox para alinhamentos, efeitos de transição e responsividade (Media Queries).

JavaScript: Manipulação do DOM para interatividade do formulário de contato (abrir/fechar modal com máscara de fundo).

🛠️ Recursos do Projeto
Menu de Navegação com Rolagem Suave: Cliques que direcionam o usuário direto para a seção desejada sem recarregar a página.

Seção de Serviços Temática: Cards estilizados com emojis e descrições divertidas inspiradas no jogo.

Formulário de Contato Dinâmico: Efeito de transição suave que desliza o formulário na tela ao clicar em "Entre em contato" ou "Fale Conosco".

Botão Flutuante do WhatsApp: Posicionamento fixo e com ajuste de camadas (z-index) para garantir o redirecionamento imediato e fácil para o usuário.
